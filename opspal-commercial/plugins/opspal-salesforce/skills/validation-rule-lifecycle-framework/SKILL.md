# Validation Rule Lifecycle Framework

Use this skill for full validation rule delivery and operations.

## Workflow

1. Define rule intent and scope.
2. Build and validate rule logic.
3. Run unit/integration/bulk tests.
4. Deploy with impact analysis and rollback readiness.
5. Monitor error trends and maintain rule health.

## Routing Boundaries

Use this skill for lifecycle operations and governance.
Use `validation-rule-patterns` for formula snippets and template lookup only.
