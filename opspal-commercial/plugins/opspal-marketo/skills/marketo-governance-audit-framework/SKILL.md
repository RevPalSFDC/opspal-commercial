# Marketo Governance Audit Framework

Use this skill for governance baselines and periodic health audits.

## Workflow

1. Validate governance baseline controls.
2. Run quarterly audit workflow and collect evidence.
3. Review automation and incident readiness controls.
4. Assess API usage and performance guardrails.
5. Produce remediation backlog with owners.

## Routing Boundaries

Use this skill for governance, audits, and policy enforcement.
Use campaign or routing diagnostics skills for active incidents.
