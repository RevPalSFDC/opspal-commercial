## Guidelines for Writing Examples

### 1. Personalized Intro Question
Match the agent's purpose:
- Discovery agents: "Need to understand your org?"
- Data agents: "Need to work with data?"
- Creation agents: "Want to create something?"
- Analysis agents: "Need to analyze X?"

### 2. Example Selection
Choose examples that:
- ✅ Cover the agent's primary use cases
- ✅ Show progression from simple → advanced
- ✅ Are copy-pastable (no placeholders users need to fill)
- ✅ Demonstrate real business value
- ❌ Don't require setup users may not have
- ❌ Don't use company-specific examples

### 3. Time Estimates
Be realistic:
- Simple queries/reads: 10-30 seconds
- Analysis/discovery: 1-3 minutes
- Creation/modification: 30 seconds - 2 minutes
- Complex operations: 2-5 minutes
- Bulk operations: 3-10 minutes

### 4. Output Descriptions
Be specific about what users get:
- ✅ "CSV file with 1,200 contacts"
- ✅ "Comprehensive audit report (PDF)"
- ✅ "Field created with confirmation"
- ❌ "Results" (too vague)
- ❌ "Output" (not descriptive)

### 5. Optional Tips
Add when relevant:
- Performance tips
- Best practices
- Common pitfalls to avoid
- When to use vs not use this agent
- Links to related agents

## Examples by Skill Level

### Beginner Examples Should:
- Use simple, clear language
- Require no prior knowledge
- Focus on common tasks everyone does
- Be immediately actionable

**Example**:
```
Use sfdc-field-analyzer to show me all fields on the Account object
```

### Intermediate Examples Should:
- Combine multiple requirements
- Show practical business scenarios
- Demonstrate filters/conditions
- Require some platform knowledge

**Example**:
```
Use sfdc-field-analyzer to find unused fields on Account that haven't been
populated in the last 90 days and are safe to delete
```

### Advanced Examples Should:
- Show complex multi-step operations
- Demonstrate edge cases
- Require deep platform knowledge
- Showcase the agent's full power

**Example**:
```
Use sfdc-field-analyzer to create a comprehensive field audit report for Account,
Contact, and Opportunity showing field usage, dependencies, and a prioritized
cleanup plan with rollback script
```
