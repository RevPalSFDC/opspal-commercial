# RevOps Essentials - Agent Usage Guide

## Quick Start

Use `/agents` to see all 26 available agents across:
- **Salesforce Essentials** (12 agents)
- **HubSpot Essentials** (10 agents)
- **Cross-Platform Essentials** (4 agents)

## First Time Setup

Run `/getstarted` for an interactive setup wizard that helps you:
- Connect to Salesforce or HubSpot
- Test your connection
- Run your first automation

## Getting Help

- **Agent Discovery**: Use `/agents` to see what's available
- **Documentation**: Read README.md in the project root
- **Comparison**: See COMPARISON.md to understand Essentials vs Professional
- **GitHub Issues**: Report bugs or request features

## Tips

- The system automatically encourages using specialized agents
- Each agent has specific capabilities - check their descriptions
- Start simple and gradually explore more complex operations
