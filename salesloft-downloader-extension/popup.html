How to Use:

  1. Ensure you're logged into Salesloft
  2. Import from Calls Page (Recommended):
    • Click "🔄 Import from Calls Page"
    • Apply your filters on the Calls page
    • Choose how many pages to extract when prompted
    • URLs will auto-populate in the extension
  3. Or manually add URLs: Paste recording URLs or load from file
  4. Click "▶️ Start Download" to begin processing
  5. Files save to: ~/Downloads/salesloft_recordings/

Features:

  • ✅ Multi-page extraction with page limit control
  • ✅ Automatic retry on connection failures
  • ✅ Resume downloads after pause or browser restart
  • ✅ Sequential processing to avoid rate limits
  • ✅ Real-time progress tracking

Tips:

  • Each page typically contains ~50 recordings
  • Processing speed: ~8-10 recordings per minute
  • Test with a few recordings first
  • Keep Chrome active during large downloads
