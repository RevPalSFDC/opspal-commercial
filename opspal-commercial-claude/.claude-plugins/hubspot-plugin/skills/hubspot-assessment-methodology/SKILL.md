# HubSpot Assessment Methodology

## When to Use This Skill

- Conducting HubSpot portal health audits
- Analyzing GTM architecture and strategy
- Evaluating automation effectiveness
- Generating executive-level assessment reports
- Benchmarking portal performance against industry standards
- Creating evidence-based recommendations

## Quick Reference

### Assessment Prompts (3 Types)

| Type | Use Case | Output |
|------|----------|--------|
| Prescriptive RevOps | Directed audit with specific questions | Structured findings + recommendations |
| Verbose Evidence-Based | Deep-dive analysis with full evidence | Comprehensive report with data citations |
| Executive Summary | High-level overview for leadership | 1-2 page strategic summary |

### Statistical Sampling Standards

| Dataset | Sample Size | Confidence |
|---------|-------------|------------|
| Contacts | 250 records | 95% |
| Deals | 100 records | 95% |
| Companies | 100 records | 95% |
| Emails | 50 campaigns | 90% |

### 7 Investigation Areas

1. **GTM Architecture** - Portal structure, object model
2. **Automation** - Workflows, sequences, triggers
3. **User Behavior** - Adoption, engagement patterns
4. **Attribution** - Source tracking, campaign ROI
5. **Data Flow** - Integration health, sync quality
6. **Reporting** - Dashboard coverage, metric accuracy
7. **Data Quality** - Completeness, accuracy, consistency
