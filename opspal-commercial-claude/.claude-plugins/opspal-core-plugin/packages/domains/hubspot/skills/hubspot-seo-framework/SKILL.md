# HubSpot SEO Framework

## When to Use This Skill

- Conducting keyword research for content strategy
- Optimizing blog posts and landing pages
- Analyzing SERP performance
- Implementing technical SEO best practices
- Building topic cluster architecture
- Optimizing for AI search engines (AEO)

## Quick Reference

### 6 Core Capabilities

| Capability | Purpose | Primary Tool |
|------------|---------|--------------|
| Keyword Research | Discover target keywords | WebSearch + HubSpot Keywords |
| Content Optimization | Improve on-page SEO | AI-powered scoring |
| SERP Analysis | Understand search landscape | Competitor analysis |
| Technical SEO | Site health and structure | HubSpot Settings |
| Topic Clusters | Content architecture | Content Strategy tool |
| AEO (AI Engine Optimization) | Optimize for AI search | Structured data + clarity |

### Content Score Components

| Component | Weight | Target |
|-----------|--------|--------|
| Keyword Usage | 25% | 1-2% density |
| Readability | 20% | Grade 8-10 |
| Meta Tags | 20% | Title < 60, Desc < 160 |
| Structure | 15% | H1, H2s, paragraphs |
| Internal Links | 10% | 3-5 per page |
| Image Optimization | 10% | Alt text, size |

### Quick Optimization Checklist

- [ ] Primary keyword in title (front-loaded)
- [ ] Primary keyword in first 100 words
- [ ] Meta description with keyword + CTA
- [ ] H1 includes primary keyword
- [ ] 3-5 internal links to related content
- [ ] Images have descriptive alt text
- [ ] URL slug includes target keyword
- [ ] Content length appropriate for topic
