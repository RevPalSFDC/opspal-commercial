# Solutions Proposal Template

Place the embedded-font PowerPoint template here:

`solutions-proposal-template.pptx`

This file should include embedded Montserrat and Figtree fonts.

## How to Create

1. Open the RevPal stylized template in PowerPoint.
2. File -> Options -> Save.
3. Enable "Embed fonts in the file".
4. Save as `solutions-proposal-template.pptx` in this folder.
