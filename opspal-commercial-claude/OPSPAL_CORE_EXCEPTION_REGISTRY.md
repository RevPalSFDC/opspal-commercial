# OpsPal Core Exception Registry

## Purpose
This registry captures temporary exceptions to Core boundary rules. Exceptions must be time-boxed, owned, and reviewed. No exception is permanent.

## Required Fields
- Exception ID
- Date Opened
- Owner
- Scope (plugin, path, or capability)
- Description
- Rationale
- Risk
- Mitigation Plan
- Expiry Date
- Review Cadence
- Approver

## Active Exceptions
None

## Retired Exceptions
None

## Template
```
Exception ID:

Scope:

```
