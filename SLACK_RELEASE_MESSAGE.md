### ⚡ **Quick Start**

```bash
git clone https://github.com/RevPalSFDC/claude-hs.git
cd claude-hs
npm install
cp .env.example .env
# Configure HubSpot credentials
npm run validate
npm run ops:console  # Launch monitoring dashboard
```
