🎙️ Salesloft Recording Download Helper

Multiple methods to download call recordings

⚠️ Prerequisites:

You must be logged into Salesloft first! Login here →

Method 1: Browser Extension (Recommended)

Install a download manager extension that can capture streaming media:

  • Chrome: Video DownloadHelper
  • Firefox: Video DownloadHelper
  • Alternative: DownloadHelper Website

Once installed, open each recording link and use the extension to download the audio.

Method 2: Save Page As (Manual)

  1. Click on a recording link above to open it
  2. Once the audio player loads, right-click on the page
  3. Select "Save Page As..." or press Ctrl+S (Windows) or Cmd+S (Mac)
  4. Choose "Webpage, Complete" as the format
  5. The audio file will be saved in the folder with the page assets

Method 3: Developer Tools

  1. Open a recording link
  2. Open Developer Tools (F12)
  3. Go to the Network tab
  4. Filter by "Media" or "Other"
  5. Play the recording
  6. Look for the actual MP3/audio file in the network requests
  7. Right-click and "Save as..."

Method 4: wget with Cookies (Advanced)

Export your browser cookies and use wget:

  1. Install a cookie export extension:
    • Get cookies.txt (Chrome)
    • cookies.txt (Firefox)
  2. While logged into Salesloft, export cookies for *.salesloft.com
  3. Save as cookies.txt in your downloads folder
  4. Run this command for each URL:
wget --load-cookies=cookies.txt --user-agent="Mozilla/5.0" \
  "https://recordings.salesloft.com/c/101314/e5332047-29cd-4ae4-abf8-206b58d9c524" \
  -O "recording_1.mp3"

Method 5: Browser Automation Script

Use this Python script with Selenium (requires Chrome/Firefox driver):

#!/usr/bin/env python3
import time
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# Recording URLs
urls = [
    "https://recordings.salesloft.com/c/101314/e5332047-29cd-4ae4-abf8-206b58d9c524",
    "https://recordings.salesloft.com/c/101314/4d3a2ec1-521f-48bc-b275-518362e85365",
    "https://recordings.salesloft.com/c/101314/8f8d6741-5f39-4f84-a581-c305db19bf70",
    "https://recordings.salesloft.com/c/101314/fe565243-a390-405c-84b3-380a3b0baf8c",
    "https://recordings.salesloft.com/c/101314/bc0fe805-4521-47b3-88db-29029c4e98d2"
]

# Setup Chrome with download preferences
options = webdriver.ChromeOptions()
prefs = {
    "download.default_directory": "/home/chris/Desktop/RevPal/Agents/iq_sdr_recordings",
    "download.prompt_for_download": False,
}
options.add_experimental_option("prefs", prefs)

driver = webdriver.Chrome(options=options)

# First, login to Salesloft
print("Please login to Salesloft in the browser window...")
driver.get("https://app.salesloft.com")
input("Press Enter after you've logged in...")

# Now download each recording
for i, url in enumerate(urls, 1):
    print(f"Opening recording {i}/{len(urls)}")
    driver.get(url)
    time.sleep(5)  # Wait for page to load
    
    # Try to find and click download button if it exists
    try:
        download_btn = driver.find_element(By.XPATH, "//button[contains(text(), 'Download')]")
        download_btn.click()
    except:
        print(f"No download button found for recording {i}, trying save page...")
        driver.execute_script("window.print();")  # Trigger print dialog as alternative
    
    time.sleep(3)

print("Done! Check your downloads folder")
driver.quit()
