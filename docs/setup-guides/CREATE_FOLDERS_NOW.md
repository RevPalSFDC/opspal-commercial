# 📁 Quick Manual Folder Setup (5 minutes)

Since the authentication is only granting read-only access, let's quickly create the folders manually.

## Open Google Drive
Go to: https://drive.google.com

## Navigate to RevPal Folder
You already have a RevPal folder with business documents.

## Create These 5 Folders Inside RevPal:

### 1. **Documentation** 📄
- Right-click in RevPal → New Folder → Name: "Documentation"
- Inside Documentation, create:
  - Salesforce
  - HubSpot
  - Integration

### 2. **Reports** 📊
- Right-click in RevPal → New Folder → Name: "Reports"
- Inside Reports, create:
  - Salesforce (then add: Daily, Weekly, Monthly)
  - HubSpot (then add: Marketing, Sales, Service)
  - Combined (then add: Executive, Operational)

### 3. **Templates** 🎨
- Right-click in RevPal → New Folder → Name: "Templates"
- Inside Templates, create:
  - Salesforce (then add: Apex, Lightning, Flows)
  - HubSpot (then add: Workflows, Emails, Forms)
  - Documentation

### 4. **Compliance** 🔒
- Right-click in RevPal → New Folder → Name: "Compliance"
- Inside Compliance, create:
  - GDPR
  - HIPAA
  - SOC2
  - CCPA

### 5. **Archives** 📦
- Right-click in RevPal → New Folder → Name: "Archives"
- Inside Archives, create:
  - 2025

## That's it! ✅

Once you create these folders (takes about 5 minutes), everything will work:
- ✅ Export reports to Google Sheets
- ✅ Access documents
- ✅ Save templates
- ✅ Archive data

## Test After Creating:
```
"List my RevPal folders"
"Export a test report to RevPal/Reports/Salesforce"
"Create a document in RevPal/Documentation"
```
