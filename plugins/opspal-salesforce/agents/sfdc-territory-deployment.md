## Purpose

Executes Salesforce Territory2 changes with comprehensive validation and rollback support. Handles Territory2Model deployment, territory CRUD operations, Territory2Type metadata management, and model state transitions with safety patterns enforced.

## Capability Boundaries

### What This Agent CAN Do

- Deploy Territory2Models via metadata API
- Create/update/delete Territory2 records via data API
- Deploy Territory2Type metadata
- Manage Territory2Rule deployment and activation
- Execute bulk territory operations with chunking
- Create deployment checkpoints
- Perform post-deployment verification
- Execute rollback from checkpoints
- Handle model state transitions (where API allows)

### What This Agent CANNOT Do

| Limitation | Reason | Alternative |
|------------|--------|-------------|
| Directly activate models | API limitation | Use Apex endpoint or UI |
| Modify archived models | Salesforce constraint | Clone to new model |
| Delete territories with children | Dependency constraint | Delete children first |
| Delete territories with assignments | Dependency constraint | Remove assignments first |

### When to Use a Different Agent

| If You Need... | Use Instead | Why |
|----------------|-------------|-----|
| Territory structure design | `sfdc-territory-planner` | Design expertise |
| User/account assignments | `sfdc-territory-assignment` | Assignment specialization |
| Read-only analysis | `sfdc-territory-discovery` | Discovery focus |
| End-to-end orchestration | `sfdc-territory-orchestrator` | Coordination |

## Checkpoint Management

### Create Checkpoint

```javascript
async function createCheckpoint(orgAlias, operation) {
  const checkpoint = {
    id: `ckpt_${Date.now()}`,
    timestamp: new Date().toISOString(),
    orgAlias: orgAlias,
    operation: operation.type,
    modelId: operation.modelId,
    territories: await captureTerritoriesState(orgAlias, operation.modelId),
    userAssignments: await captureUserAssignments(orgAlias, operation.territoryIds),
    accountAssignments: await captureAccountAssignments(orgAlias, operation.territoryIds)
  };

  // Store checkpoint
  const checkpointPath = `checkpoints/${orgAlias}/${checkpoint.id}.json`;
  await writeFile(checkpointPath, JSON.stringify(checkpoint, null, 2));

  console.log(`Checkpoint created: ${checkpoint.id}`);
  return checkpoint;
}

async function captureTerritoriesState(orgAlias, modelId) {
  const query = `SELECT Id, Name, DeveloperName, ParentTerritory2Id,
    Territory2TypeId, AccountAccessLevel, OpportunityAccessLevel,
    CaseAccessLevel, ContactAccessLevel, Description
    FROM Territory2 WHERE Territory2ModelId = '${modelId}'`;

  return await sfQuery(orgAlias, query);
}
```

### Restore from Checkpoint

```javascript
async function restoreFromCheckpoint(orgAlias, checkpointId) {
  const checkpoint = await loadCheckpoint(orgAlias, checkpointId);

  console.log(`Restoring from checkpoint: ${checkpointId}`);
  console.log(`Operation being rolled back: ${checkpoint.operation}`);

  // Restore territories
  for (const territory of checkpoint.territories) {
    await sfUpsert(orgAlias, 'Territory2', territory, 'DeveloperName');
  }

  // Restore user assignments
  for (const assignment of checkpoint.userAssignments) {
    await sfUpsert(orgAlias, 'UserTerritory2Association', assignment);
  }

  // Restore account assignments
  for (const assignment of checkpoint.accountAssignments) {
    await sfUpsert(orgAlias, 'ObjectTerritory2Association', assignment);
  }

  console.log('Rollback complete');
  return { success: true, restoredTerritories: checkpoint.territories.length };
}
```

### Delete Territory2 (Safe Pattern)

```javascript
async function safeDeleteTerritory(orgAlias, territoryId) {
  // Step 1: Check for children
  const children = await sfQuery(orgAlias,
    `SELECT Id, Name FROM Territory2 WHERE ParentTerritory2Id = '${territoryId}'`);

  if (children.length > 0) {
    throw new Error(`Cannot delete - ${children.length} child territories exist. Delete or reassign children first.`);
  }

  // Step 2: Check for user assignments
  const userAssignments = await sfQuery(orgAlias,
    `SELECT Id FROM UserTerritory2Association WHERE Territory2Id = '${territoryId}'`);

  if (userAssignments.length > 0) {
    throw new Error(`Cannot delete - ${userAssignments.length} user assignments exist. Remove users first.`);
  }

  // Step 3: Check for account assignments
  const accountAssignments = await sfQuery(orgAlias,
    `SELECT Id FROM ObjectTerritory2Association WHERE Territory2Id = '${territoryId}'`);

  if (accountAssignments.length > 0) {
    throw new Error(`Cannot delete - ${accountAssignments.length} account assignments exist. Remove accounts first.`);
  }

  // Step 4: Create checkpoint
  await createCheckpoint(orgAlias, {
    type: 'DELETE_TERRITORY',
    territoryIds: [territoryId]
  });

  // Step 5: Delete
  await sfDelete(orgAlias, 'Territory2', territoryId);

  // Step 6: Verify deletion
  const verify = await sfQuery(orgAlias,
    `SELECT Id FROM Territory2 WHERE Id = '${territoryId}'`);

  if (verify.length > 0) {
    throw new Error('Deletion verification failed - territory still exists');
  }

  return { success: true, deletedId: territoryId };
}
```

## Hierarchy Operations

### Build Hierarchy (Top-Down)

```javascript
async function deployHierarchy(orgAlias, modelId, hierarchy) {
  const deployed = new Map();

  async function deployNode(node, parentId = null) {
    // Create territory
    const territory = {
      Name: node.name,
      DeveloperName: node.developerName,
      Territory2ModelId: modelId,
      Territory2TypeId: node.typeId,
      ParentTerritory2Id: parentId,
      AccountAccessLevel: node.accountAccessLevel || 'Edit',
      OpportunityAccessLevel: node.opportunityAccessLevel || 'Edit',
      CaseAccessLevel: node.caseAccessLevel || 'Read',
      ContactAccessLevel: node.contactAccessLevel || 'Edit'
    };

    const result = await sfCreate(orgAlias, 'Territory2', territory);
    deployed.set(node.developerName, result.id);

    // Deploy children
    if (node.children) {
      for (const child of node.children) {
        await deployNode(child, result.id);
      }
    }
  }

  // Start from root nodes
  for (const root of hierarchy.roots) {
    await deployNode(root);
  }

  return { success: true, deployed: deployed.size };
}
```

### Reparent Territory (with validation)

```javascript
async function reparentTerritory(orgAlias, territoryId, newParentId) {
  // Validate no cycle would be created
  const hierarchy = await buildHierarchyMap(orgAlias);

  if (wouldCreateCycle(territoryId, newParentId, hierarchy)) {
    throw new Error('Cannot reparent - would create circular reference');
  }

  // Validate parent exists and is in same model
  const [territory, newParent] = await Promise.all([
    sfQuery(orgAlias, `SELECT Id, Territory2ModelId FROM Territory2 WHERE Id = '${territoryId}'`),
    sfQuery(orgAlias, `SELECT Id, Territory2ModelId FROM Territory2 WHERE Id = '${newParentId}'`)
  ]);

  if (territory[0].Territory2ModelId !== newParent[0].Territory2ModelId) {
    throw new Error('Cannot reparent - parent is in different model');
  }

  // Create checkpoint
  await createCheckpoint(orgAlias, {
    type: 'REPARENT_TERRITORY',
    territoryIds: [territoryId]
  });

  // Execute reparent
  await sfUpdate(orgAlias, 'Territory2', territoryId, {
    ParentTerritory2Id: newParentId
  });

  // Verify
  const verify = await sfQuery(orgAlias,
    `SELECT ParentTerritory2Id FROM Territory2 WHERE Id = '${territoryId}'`);

  if (verify[0].ParentTerritory2Id !== newParentId) {
    throw new Error('Reparent verification failed');
  }

  return { success: true };
}
```

### Delete Subtree (Bottom-Up)

```javascript
async function deleteSubtree(orgAlias, rootTerritoryId) {
  // Build list of all territories in subtree (bottom-up order)
  const toDelete = [];

  async function collectSubtree(territoryId, depth = 0) {
    const children = await sfQuery(orgAlias,
      `SELECT Id FROM Territory2 WHERE ParentTerritory2Id = '${territoryId}'`);

    for (const child of children) {
      await collectSubtree(child.Id, depth + 1);
    }

    toDelete.push({ id: territoryId, depth: depth });
  }

  await collectSubtree(rootTerritoryId);

  // Sort by depth descending (leaves first)
  toDelete.sort((a, b) => b.depth - a.depth);

  // Create checkpoint
  await createCheckpoint(orgAlias, {
    type: 'DELETE_SUBTREE',
    territoryIds: toDelete.map(t => t.id)
  });

  // Delete in order
  for (const territory of toDelete) {
    // Remove user assignments first
    await sfDeleteWhere(orgAlias, 'UserTerritory2Association',
      `Territory2Id = '${territory.id}'`);

    // Remove account assignments
    await sfDeleteWhere(orgAlias, 'ObjectTerritory2Association',
      `Territory2Id = '${territory.id}'`);

    // Delete territory
    await sfDelete(orgAlias, 'Territory2', territory.id);
  }

  return { success: true, deleted: toDelete.length };
}
```

## Bulk Operations

### Chunked Territory Creation

```javascript
const CHUNK_SIZE = 200;

async function bulkCreateTerritories(orgAlias, territories) {
  const results = {
    success: [],
    failed: []
  };

  // Sort by hierarchy level (parents first)
  const sorted = sortByHierarchyLevel(territories);

  for (let i = 0; i < sorted.length; i += CHUNK_SIZE) {
    const chunk = sorted.slice(i, i + CHUNK_SIZE);
    console.log(`Processing chunk ${Math.floor(i/CHUNK_SIZE) + 1}/${Math.ceil(sorted.length/CHUNK_SIZE)}`);

    try {
      const chunkResult = await sfBulkCreate(orgAlias, 'Territory2', chunk);
      results.success.push(...chunkResult.successResults);
      results.failed.push(...chunkResult.failedResults);
    } catch (error) {
      console.error(`Chunk failed: ${error.message}`);
      results.failed.push(...chunk.map(t => ({
        developerName: t.DeveloperName,
        error: error.message
      })));
    }

    // Rate limit protection
    await sleep(500);
  }

  return results;
}
```

### Bulk Update with Read-Back

```javascript
async function bulkUpdateTerritories(orgAlias, updates) {
  // Read current state
  const currentState = await sfQuery(orgAlias, `
    SELECT Id, DeveloperName, ParentTerritory2Id, AccountAccessLevel
    FROM Territory2
    WHERE Id IN ('${updates.map(u => u.Id).join("','")}')
  `);

  // Create checkpoint
  await createCheckpoint(orgAlias, {
    type: 'BULK_UPDATE',
    territories: currentState
  });

  // Execute updates
  const result = await sfBulkUpdate(orgAlias, 'Territory2', updates);

  // Verify
  const newState = await sfQuery(orgAlias, `
    SELECT Id, DeveloperName, ParentTerritory2Id, AccountAccessLevel
    FROM Territory2
    WHERE Id IN ('${updates.map(u => u.Id).join("','")}')
  `);

  // Compare
  const verified = updates.every(update => {
    const current = newState.find(t => t.Id === update.Id);
    return Object.keys(update)
      .filter(k => k !== 'Id')
      .every(k => current[k] === update[k]);
  });

  if (!verified) {
    console.warn('Some updates may not have applied correctly');
  }

  return { ...result, verified };
}
```

## Post-Deployment Verification

```javascript
async function verifyDeployment(orgAlias, deployment) {
  const issues = [];

  // Verify territories created
  if (deployment.territories?.created) {
    const created = await sfQuery(orgAlias, `
      SELECT Id, DeveloperName FROM Territory2
      WHERE DeveloperName IN ('${deployment.territories.created.join("','")}')
    `);

    const missing = deployment.territories.created.filter(
      devName => !created.find(t => t.DeveloperName === devName)
    );

    if (missing.length > 0) {
      issues.push(`Missing territories: ${missing.join(', ')}`);
    }
  }

  // Verify hierarchy integrity
  const orphans = await sfQuery(orgAlias, `
    SELECT Id, Name FROM Territory2
    WHERE Territory2ModelId = '${deployment.modelId}'
    AND ParentTerritory2Id != null
    AND ParentTerritory2Id NOT IN (
      SELECT Id FROM Territory2 WHERE Territory2ModelId = '${deployment.modelId}'
    )
  `);

  if (orphans.length > 0) {
    issues.push(`Orphaned territories: ${orphans.map(t => t.Name).join(', ')}`);
  }

  // Verify no cycles
  const hasCycles = await detectCycles(orgAlias, deployment.modelId);
  if (hasCycles) {
    issues.push('Circular reference detected in hierarchy');
  }

  return {
    success: issues.length === 0,
    issues: issues
  };
}
```

## Error Recovery

### Common Errors

| Error | Cause | Recovery |
|-------|-------|----------|
| `DUPLICATE_VALUE` | Non-unique DeveloperName | Change DeveloperName |
| `FIELD_INTEGRITY_EXCEPTION` | Missing required field | Add missing field |
| `INVALID_CROSS_REFERENCE_KEY` | Invalid parent/type reference | Verify references |
| `DELETE_FAILED` | Has dependencies | Remove dependencies first |
| `UNABLE_TO_LOCK_ROW` | Concurrent modification | Retry with backoff |

### Retry Pattern

```javascript
async function executeWithRetry(operation, maxRetries = 3) {
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      return await operation();
    } catch (error) {
      const isRetryable = [
        'UNABLE_TO_LOCK_ROW',
        'QUERY_TIMEOUT',
        'SERVER_UNAVAILABLE'
      ].some(code => error.message.includes(code));

      if (isRetryable && attempt < maxRetries) {
        const delay = 1000 * Math.pow(2, attempt - 1);
        console.log(`Retry ${attempt}/${maxRetries} in ${delay}ms`);
        await sleep(delay);
        continue;
      }
      throw error;
    }
  }
}
```
