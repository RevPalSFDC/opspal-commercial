## Goal Properties

HubSpot Goal Targets expose these key properties:

| Property | Description | Type |
|----------|-------------|------|
| `hs_goal_name` | Display name for the goal | String |
| `hs_target_amount` | Quota/target value | Number |
| `hs_start_datetime` | Goal period start | DateTime |
| `hs_end_datetime` | Goal period end | DateTime |
| `hs_created_by_user_id` | Goal owner/assignee | String |

## Common Tasks

### Task 1: List All Active Goals

```javascript
const manager = new HubSpotGoalsManager(accessToken);
const activeGoals = await manager.getActiveGoals();

console.log(`Found ${activeGoals.length} active goals`);
activeGoals.forEach(goal => {
  console.log(`- ${goal.properties.hs_goal_name}: $${goal.properties.hs_target_amount}`);
});
```

### Task 2: Get User's Goals

```javascript
const userGoals = await manager.getUserGoals('user-id-123');

userGoals.forEach(goal => {
  console.log(`Goal: ${goal.properties.hs_goal_name}`);
  console.log(`  Target: $${goal.properties.hs_target_amount}`);
  console.log(`  Period: ${goal.properties.hs_start_datetime} - ${goal.properties.hs_end_datetime}`);
});
```

### Task 3: Calculate Goal Progress

```javascript
// actualRevenue comes from deals analysis
const actualRevenue = 75000;
const progress = await manager.calculateGoalProgress('goal-id-456', actualRevenue);

console.log(`Goal: ${progress.goalName}`);
console.log(`  Target: $${progress.target}`);
console.log(`  Actual: $${progress.actual}`);
console.log(`  Attainment: ${progress.attainmentPercent}%`);
console.log(`  Status: ${progress.status}`);
console.log(`  Days Remaining: ${progress.daysRemaining}`);
console.log(`  Required Daily: $${progress.requiredDailyPace}`);
```

### Task 4: Generate Goals Summary Report

```javascript
const goals = await manager.getAllGoals();

// actualAmounts would come from deal/revenue analysis
const actualAmounts = new Map([
  ['goal-1', 50000],
  ['goal-2', 120000],
  ['goal-3', 30000]
]);

const summary = manager.generateGoalsSummary(goals, actualAmounts);

console.log('=== Goals Summary Report ===');
console.log(`Total Goals: ${summary.totalGoals}`);
console.log(`Active Goals: ${summary.activeGoals}`);
console.log(`Overall Attainment: ${summary.overallAttainment}%`);
console.log('');
console.log('Status Breakdown:');
Object.entries(summary.statusBreakdown).forEach(([status, count]) => {
  if (count > 0) console.log(`  ${status}: ${count}`);
});
```

### Task 5: Get Q1 Goals

## Goal Status Definitions

| Status | Description | Criteria |
|--------|-------------|----------|
| `ACHIEVED` | Goal met or exceeded | Attainment >= 100% |
| `ON_TRACK` | Pace suggests goal will be met | Attainment >= 90% of expected pace |
| `AT_RISK` | May miss goal without intervention | Attainment 70-90% of expected pace |
| `BEHIND` | Significantly off pace | Attainment < 70% of expected pace |
| `MISSED` | Goal period ended without achievement | Period ended, attainment < 100% |
| `UNKNOWN` | Cannot determine status | Missing date information |

## Error Handling

### Common Issues

| Error | Cause | Resolution |
|-------|-------|------------|
| `401 Unauthorized` | Invalid or expired token | Refresh access token |
| `403 Forbidden` | Insufficient permissions | Check API key scopes |
| `404 Not Found` | Goal ID doesn't exist | Verify goal ID |
| `429 Rate Limit` | Too many requests | Automatic retry with backoff |
| `500 Server Error` | HubSpot API issue | Retry after delay |

### Validation Checklist

- [ ] Access token has `crm.objects.goals.read` scope
- [ ] Goal IDs are valid HubSpot record IDs
- [ ] Date filters use ISO 8601 format
- [ ] User IDs match HubSpot user records
- [ ] Actual amounts are numeric values

### Caching Strategy
```javascript
// Cache goals for 5 minutes to reduce API calls
const CACHE_TTL = 5 * 60 * 1000;
let goalsCache = { data: null, timestamp: 0 };

async function getCachedGoals() {
  if (Date.now() - goalsCache.timestamp < CACHE_TTL) {
    return goalsCache.data;
  }
  goalsCache.data = await manager.getAllGoals();
  goalsCache.timestamp = Date.now();
  return goalsCache.data;
}
```

### Progress Tracking Pattern
```javascript
// Track progress over time
async function trackGoalProgress(goalId) {
  const goal = await manager.getGoal(goalId);
  const deals = await getClosedDealsForGoal(goal);
  const actualAmount = deals.reduce((sum, d) => sum + d.amount, 0);

  return manager.calculateGoalProgress(goalId, actualAmount);
}
```
