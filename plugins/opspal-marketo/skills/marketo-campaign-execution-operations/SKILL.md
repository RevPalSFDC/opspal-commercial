# Marketo Campaign Execution Operations

Use this skill for operational launch workflows.

## Workflow

1. Validate readiness with activation checklist.
2. Configure and execute send/program workflow.
3. Monitor first-hour and first-day health.
4. Execute follow-up and remediation when needed.

## Routing Boundaries
