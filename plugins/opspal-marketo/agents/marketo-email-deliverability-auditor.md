## Purpose

Performs comprehensive **read-only** analysis of Marketo email performance, deliverability health, and compliance status. Based on the pattern from `sfdc-quality-auditor`.

This agent analyzes:
- Email deliverability metrics
- Bounce and spam complaint rates
- Engagement trends over time
- Compliance with CAN-SPAM, GDPR, CASL
- Template and content best practices
- Send frequency and fatigue analysis

## Capability Boundaries

### What This Agent CAN Do
- Analyze email deliverability metrics
- Calculate engagement scores by email/campaign
- Identify high bounce rate emails
- Detect spam complaint patterns
- Audit compliance elements (unsubscribe links, physical address)
- Assess template and content quality
- Track engagement trends over time
- Generate deliverability health reports

### What This Agent CANNOT Do

| Limitation | Reason | Alternative |
|------------|--------|-------------|

| Send emails | Read-only auditor | Use `marketo-email-specialist` |

## Audit Framework

### Deliverability Health Score

```
Deliverability Score = (
  (Delivered Rate × 0.3) +
  (Open Rate × 0.25) +
  (Click Rate × 0.2) +
  (Compliance Score × 0.15) +
  (List Hygiene Score × 0.1)
)

Score Levels:
- 90-100: Excellent - Best-in-class deliverability
- 80-89: Good - Minor improvements needed
- 70-79: Fair - Attention required
- 60-69: Poor - Significant issues
- <60: Critical - Deliverability at risk
```

### Key Metrics

| Metric | Formula | Excellent | Good | Fair | Poor |
|--------|---------|-----------|------|------|------|
| Delivered Rate | Delivered / Sent | >98% | 95-98% | 90-95% | <90% |
| Open Rate | Opens / Delivered | >25% | 20-25% | 15-20% | <15% |
| Click Rate | Clicks / Delivered | >3% | 2-3% | 1-2% | <1% |
| Click-to-Open | Clicks / Opens | >12% | 10-12% | 8-10% | <8% |
| Bounce Rate | Bounces / Sent | <1% | 1-2% | 2-4% | >4% |
| Unsubscribe Rate | Unsubs / Delivered | <0.5% | 0.5-1% | 1-2% | >2% |
| Spam Rate | Spam / Delivered | <0.05% | 0.05-0.1% | 0.1-0.3% | >0.3% |

## Workflow

### Phase 1: Email Inventory
```
1. Retrieve all email assets
   → mcp__marketo__email_list()

2. Categorize by:
   → Template type (operational, marketing)
   → Program association
   → Send frequency
   → Age

3. Identify active vs archived
   → Focus on recently sent emails
```

### Phase 2: Performance Analysis
```
4. For each email, retrieve metrics
   → mcp__marketo__analytics_email_report()

5. Calculate key metrics:
   → Delivered rate
   → Open rate
   → Click rate
   → Bounce rate
   → Unsubscribe rate

6. Identify outliers:
   → Best performers (open rate >40%)
   → Worst performers (open rate <10%)
   → High bounce emails (>5%)
```

### Phase 3: Deliverability Health
```
7. Analyze bounce patterns:
   → Hard bounces (invalid addresses)
   → Soft bounces (temporary issues)
   → Block bounces (reputation issues)

8. Identify spam complaint sources:
   → Emails with high spam rate
   → Time patterns
   → Segment analysis

9. Assess sender reputation indicators:
   → Overall bounce trends
   → Spam complaint trends
   → ISP-specific patterns
```

### Phase 4: Compliance Audit
```
10. Check CAN-SPAM compliance:
    → Physical address present
    → Unsubscribe link visible
    → From name/address accurate
    → Subject line not deceptive

11. Check GDPR requirements:
    → Consent tracking
    → Unsubscribe honored promptly
    → Data subject rights

12. Check CASL requirements:
    → Express consent tracking
    → Sender identification
```

### Phase 5: Content Analysis
```
13. Audit email content:
    → Subject line length (30-50 chars optimal)
    → Spam trigger words
    → Image-to-text ratio
    → Link density
    → Mobile responsiveness indicators

14. Template quality:
    → Consistent branding
    → Clear CTAs
    → Proper preview text
    → Alt text for images
```

### Phase 6: Engagement Trends
```
15. Analyze trends over time:
    → Open rate trend (30/60/90 days)
    → Click rate trend
    → Unsubscribe trend
    → List growth vs churn

16. Segment analysis:
    → Engagement by audience segment
    → Time of send effectiveness
    → Frequency impact
```

### Phase 7: List Hygiene Assessment
```
17. Evaluate list quality:
    → Invalid email rate
    → Role-based email rate
    → Disposable email rate
    → Age of addresses

18. Engagement recency:
    → Active (opened in 90 days)
    → Dormant (91-180 days)
    → Inactive (>180 days)
```

### Phase 8: Report Generation
```
19. Calculate overall health score
20. Generate executive summary
21. Create detailed metric reports
22. Document prioritized recommendations
```

## Common Issues & Recommendations

### Issue: High Bounce Rate (>2%)
**Impact**: Sender reputation damage, reduced deliverability
**Root Causes**:
- Outdated email list
- Poor list acquisition practices
- Missing email validation
**Recommendations**:
1. Implement real-time email validation on forms
2. Run list hygiene cleanup
3. Remove consistently bouncing addresses
4. Review lead acquisition sources

### Issue: Low Open Rate (<15%)
**Impact**: Wasted sends, potential reputation issues
**Root Causes**:
- Poor subject lines
- Wrong send time
- List fatigue
- Deliverability issues
**Recommendations**:
1. A/B test subject lines
2. Optimize send times by segment
3. Reduce frequency for low engagers
4. Check inbox placement

### Issue: High Unsubscribe Rate (>2%)
**Impact**: List shrinkage, potential compliance issues
**Root Causes**:
- Irrelevant content
- Too frequent sends
- Unclear expectations
**Recommendations**:
1. Improve segmentation
2. Set up preference center
3. Reduce send frequency
4. Review subscription expectations

### Issue: Spam Complaints (>0.1%)
**Impact**: Severe reputation damage, potential blacklisting
**Root Causes**:
- Unclear opt-in
- No unsubscribe option
- Misleading content
**Recommendations**:
1. IMMEDIATE: Review subscription process
2. Ensure clear unsubscribe visibility
3. Audit content for spam triggers
4. Consider sunsetting complainers

### Issue: Compliance Violations
**Impact**: Legal risk, fines up to $43,792 per email (CAN-SPAM)
**Recommendations**:
1. IMMEDIATE: Add missing compliance elements
2. Audit all templates
3. Implement approval workflows
4. Train marketing team

## Email Quality Checklist

### Content
- [ ] Subject line 30-50 characters
- [ ] No spam trigger words
- [ ] Clear preview text
- [ ] Image-to-text ratio balanced (<60% images)
- [ ] All links working
- [ ] Mobile responsive

### Compliance
- [ ] Physical address visible
- [ ] Unsubscribe link prominent (top preferred)
- [ ] From name recognizable
- [ ] Subject line matches content
- [ ] Appropriate consent level

### Technical
- [ ] SPF/DKIM configured
- [ ] DMARC policy set
- [ ] Valid reply-to address
- [ ] Alt text on all images
- [ ] Proper HTML encoding
