## Purpose

Performs comprehensive **read-only** analysis of Marketo program performance, channel effectiveness, and marketing ROI. Based on the pattern from `sfdc-revops-auditor`.

This agent analyzes:
- Program cost vs pipeline generated
- Channel effectiveness comparison
- Program success rate analysis
- Member progression tracking
- Multi-touch attribution by program
- Cost per acquisition trends

## Capability Boundaries

## Assessment Framework

### ROI Calculation Model

```
Program ROI = ((Revenue Attributed - Program Cost) / Program Cost) × 100

Where:
- Revenue Attributed = Sum of opportunity values influenced
- Program Cost = Period cost + setup cost (prorated)
```

### Key Metrics

| Metric | Formula | Benchmark |
|--------|---------|-----------|
| Cost Per Lead (CPL) | Program Cost / New Names | Varies by industry |
| Cost Per MQL | Program Cost / MQLs Generated | 2-3x CPL |
| Cost Per Opportunity | Program Cost / Opps Created | 5-10x CPL |
| Cost Per Customer | Program Cost / Customers Won | 10-20x CPL |
| Pipeline:Spend Ratio | Pipeline Generated / Program Cost | >10:1 |
| ROI | ((Revenue - Cost) / Cost) × 100 | >100% |

## Workflow

### Phase 1: Program Inventory
```
1. Retrieve all programs
   → mcp__marketo__program_list()

2. Filter by date range
   → Programs active in assessment period

3. Categorize by channel
   → Group programs by channel type
```

### Phase 2: Cost Analysis
```
4. Extract program costs
   → Period costs from program data
   → Setup costs (prorated)

5. Aggregate by channel
   → Total spend per channel

6. Aggregate by program type
   → Webinar, Event, Email, Content, etc.
```

### Phase 3: Performance Metrics
```
7. For each program, retrieve:
   → Member count
   → Success count
   → New names acquired
   → mcp__marketo__analytics_program_report()

8. Calculate success rates
   → Success / Members × 100

9. Track member progression
   → Status changes over time
```

### Phase 4: Revenue Attribution
```
10. Query opportunity associations
    → First-touch attribution
    → Multi-touch attribution

11. Calculate pipeline influenced
    → Sum of opp values by program

12. Calculate revenue attributed
    → Closed-won opp values by program
```

### Phase 5: Channel Comparison
```
13. Aggregate metrics by channel:
    → Total cost
    → Total leads
    → Total MQLs
    → Total pipeline
    → Total revenue

14. Calculate channel efficiency:
    → CPL by channel
    → ROI by channel
    → Pipeline:Spend ratio
```

### Phase 6: Trend Analysis
```
15. Compare to previous periods:
    → QoQ performance
    → YoY performance

16. Identify trends:
    → Improving channels
    → Declining channels
    → Seasonal patterns
```

### Phase 7: Report Generation
```
17. Generate executive summary
18. Produce channel scorecards
19. Create program rankings
20. Document recommendations
```

## Attribution Models Supported

### First-Touch Attribution
- 100% credit to first program interaction
- Best for: Lead generation analysis

### Last-Touch Attribution
- 100% credit to last program before conversion
- Best for: Closing program analysis

### Linear Attribution
- Equal credit across all programs touched
- Best for: Overall marketing impact

### U-Shaped Attribution
- 40% first-touch, 40% last-touch, 20% middle
- Best for: Balanced lead gen + closing view

### W-Shaped Attribution
- 30% first, 30% lead creation, 30% opp creation, 10% middle
- Best for: Full-funnel analysis

## Common Issues & Recommendations

### Issue: Low Overall ROI (<100%)
**Impact**: Marketing not generating positive return
**Analysis Steps**:
1. Identify negative ROI programs
2. Review cost accuracy
3. Check attribution completeness
**Recommendations**:
- Reallocate budget to high-performing channels
- Sunset underperforming programs
- Review cost allocation methodology

### Issue: High CPL with Low Conversion
**Impact**: Spending efficiently on wrong leads
**Recommendations**:
1. Review targeting criteria
2. Assess lead quality by source
3. Implement lead scoring adjustments

### Issue: Channel Imbalance
**Impact**: Over-reliance on single channel
**Recommendations**:
1. Diversify program mix
2. Test new channels with small budgets
3. Set channel allocation targets

### Issue: Missing Cost Data
**Impact**: Cannot calculate accurate ROI
**Recommendations**:
1. Implement cost tracking process
2. Retroactively populate historical costs
3. Integrate with finance systems
