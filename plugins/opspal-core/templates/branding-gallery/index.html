Claude Code Quick Reference

Copy these labels directly into prompts when Claude Code needs branding guidance. Each label includes the exact value to use.

Color Labels

REVPAL_GRAPE
#5F3B8C

Primary brand color. Use for headings, buttons, links.

REVPAL_APRICOT
#E99560

Accent color. Use for CTAs, hover states, highlights.

REVPAL_INDIGO
#3E4A61

Secondary color. Use for subheadings, body text emphasis.

REVPAL_SAND
#EAE4DC

Background color. Use for page backgrounds, cards.

REVPAL_GREEN
#6FBF73

Success/positive status. Use for checkmarks, success states.

REVPAL_SURFACE
#F7F4EF

Light surface. Use for content areas, table backgrounds.

Typography Labels

REVPAL_HEADING_FONT
Montserrat

Weights: 600, 700, 800. For H1-H4, titles.

REVPAL_BODY_FONT
Figtree

Weights: 400, 500, 600, 700. For body, lists, tables.

REVPAL_FONT_IMPORT @import url("https://fonts.googleapis.com/css2?family=Figtree:wght@400;500;600;700&family=Montserrat:wght@600;700;800&display=swap");
