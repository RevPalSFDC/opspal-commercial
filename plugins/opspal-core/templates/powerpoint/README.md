# PowerPoint Template Library

This folder is the central library for PowerPoint templates used by the
OpsPal Core plugin.

## Default Template

- **Solutions Proposal**
  - Path: `templates/powerpoint/solutions-proposal/solutions-proposal-template.pptx`
  - Purpose: Stylized RevPal solutions proposal template with embedded fonts.

## Manifest

See `templates/powerpoint/manifest.json` for the canonical template list.

## Legacy Compatibility

If the new template path is missing, the generator will fall back to the
legacy location: `templates/pptx-templates/revpal-master.pptx`.
