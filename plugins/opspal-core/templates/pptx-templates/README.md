# PPTX Templates (Legacy)

This folder is kept for backwards compatibility. The canonical PowerPoint
template library now lives under:

`templates/powerpoint/`

## Legacy Template Path

If you still use the old path, place a PowerPoint file with embedded
Montserrat and Figtree fonts at:

`templates/pptx-templates/revpal-master.pptx`

The PPTX generator will fall back to this location when the new library
template is missing.
